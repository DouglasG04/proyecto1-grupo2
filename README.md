## 📂 Project Structure

```bash

├── src/                  
│   ├── public/            # Frontend
│   ├── server/            # Backend

│       ├── utils

```

## 💻 Developing Locally

### 1. Clone the Repository
```bash 
git clone https://github.com/DouglasG04/proyecto1-grupo2.git
```

### 2. Install Dependencies
```bash 
npm install
```

### 3. Start server (Note: DB is not deployed in an external service)
```bash 
npm start
```
